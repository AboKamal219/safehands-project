## 🚀 Installation & Setup
### 1️⃣ Download XAMPP
First, download and install XAMPP from the official website:
👉 https://www.apachefriends.org/download.html
في البداية قم بتحميل برنامج XAMPP من الموقع الرسمي من خلال الرابط الموجود بالأعلى.
---
### 2️⃣ Move Project Files
After downloading the project, move it to the following path:
```
C:\xampp\htdocs
```
🔹 This step is important because this folder is the default directory for XAMPP server.
Putting your project here allows it to run directly through the local server.
بعد تحميل المشروع، قم بنقله داخل المسار الموضح بالأعلى، لأن هذا المجلد هو المسؤول عن تشغيل المشاريع على السيرفر المحلي، وبالتالي سيفتح المشروع مباشرة.

3️⃣ Run the Project
Start **Apache** from XAMPP Control Panel, then open your browser and go to:
```
http://localhost/your-project-folder    اسم مجلد المشروع
```
قم بتشغيل Apache من داخل XAMPP، ثم افتح المتصفح واكتب الرابط بالأعلى مع اسم مجلد المشروع لفتح الموقع.
